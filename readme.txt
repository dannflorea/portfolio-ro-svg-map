Display an elegant portfolio with filters by category and interactive SVG map of Romania. Includes infinite scroll, AJAX loading, and Elementor compatibility.

== Description ==

📍 **WizeWP Portfolio Romania Map** is a lightweight, flexible plugin that allows you to showcase WordPress portfolio items filtered by:

- County (via an interactive SVG map of Romania)
- Custom categories
- AJAX-based filters (no page reload)
- Infinite scroll
- Responsive, animated grid layout
- Elementor & shortcode integration

This plugin is ideal for showcasing projects with a geographic footprint — grants, case studies, service providers, etc.

### ✅ Features

- 🗺️ Interactive SVG map of Romania (clickable counties)
- 🎯 Filter portfolio items by region & category
- 🔁 Infinite scroll (AJAX powered)
- 💡 Hover overlay with project details
- 🧩 Easy integration via shortcode or Elementor
- 🧱 Grid layout (4 columns desktop, 2 columns mobile)
- 📱 Mobile responsive & optimized

== Installation ==

1. Download and extract the plugin.
2. Upload the folder to `/wp-content/plugins/interactive-portfolio-map/`.
3. Activate the plugin from the WordPress Dashboard.
4. Use the shortcode `[wprm_portfolio_map]` in any page, post or Elementor section.

== Usage ==

- Go to **Portfolio > Add New** to create your items.
- Assign **categories** and **counties** (custom taxonomy).
- Insert the `[wprm_portfolio_map]` shortcode on your desired page.
- That's it!

== Shortcode ==

`[wprm_portfolio_map]` 

== Frequently Asked Questions ==

= Can I use this plugin with Elementor? =  
Yes, just insert the shortcode inside a Shortcode widget in Elementor.

= Is the map clickable? =  
Yes, each county is interactive and highlights on click.

= Can I change the colors or layout? =  
You can override styles via your theme or child theme. We’ll add admin options in a future release.
